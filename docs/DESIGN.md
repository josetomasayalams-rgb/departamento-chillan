# Diseño

La interfaz prioriza lectura rápida del calendario y contraste de las familias. Liquid Glass es un lenguaje visual, no una razón para sacrificar legibilidad, interacción táctil o contenido en español.

El calendario usa una ventana móvil de 31 días consecutivos desde hoy, en vez de un mes calendario fijo. El encabezado comunica el rango completo, la grilla marca el cambio de mes y el periodo avanza al cambiar la fecha local. La navegación manual se conserva hasta tocar `Desde hoy`. El margen de Airbnb se recalcula con el mismo cambio diario.
